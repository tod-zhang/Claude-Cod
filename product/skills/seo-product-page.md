# Skill: SEO Product Page Creator

## Trigger
`/create-product-page [product-name]`

## Description
Creates a comprehensive, SEO-optimized product page for metal-castings.com based on competitive analysis and differentiation strategy.

## Workflow

### Step 1: Competitive Research
When triggered, execute the following searches:

```
Search queries:
1. "[product-name] manufacturer supplier"
2. "[product-name] custom machining casting"
3. "[product-name] specifications technical"
4. "[product-name] applications industries"
```

### Step 2: Analyze Top 3 Competitors
For each competitor, extract:
- Page structure (H1-H3)
- Key content sections
- Trust signals
- Unique selling points
- Content gaps

### Step 3: Identify Differentiation Opportunities
Based on analysis, determine:

**Content Gaps:**
- Topics competitors don't cover
- Questions left unanswered
- Technical details missing

**Quality Gaps:**
- Better images needed
- Deeper technical content
- More comprehensive specs

**Trust Gaps:**
- Missing certifications
- No social proof
- Lack of expertise signals

### Step 4: Generate Page Content

#### Title Tag Template
```
Custom [Product Name] | [Key Differentiator] | Metal-Castings.com
```

#### Meta Description Template
```
Looking for high-quality custom [product name]? Metal-Castings.com offers [differentiator]. Get a quote for your [product] project today. [Certification] certified.
```

#### Content Sections to Generate

1. **Hero Section**
   - H1 with primary keyword
   - 2-sentence value proposition
   - Primary CTA

2. **Product Overview** (300-400 words)
   - What is [product]
   - Key benefits
   - Primary applications

3. **Specifications Table**
   - Materials available
   - Size ranges
   - Tolerances
   - Finishes

4. **Manufacturing Process** (200-300 words)
   - Our process steps
   - Quality control
   - Capabilities unique to us

5. **Industries & Applications** (400-500 words)
   - 4-6 industry applications
   - Specific use cases
   - Problem-solution format

6. **Why Choose Us** (200-300 words)
   - Key differentiators
   - Certifications
   - Experience/credentials

7. **FAQ Section** (5-8 questions)
   - Address common buyer questions

8. **CTA Section**
   - Contact form
   - Phone number
   - Response time promise

### Step 5: Create Asset List
Recommend:
- Product images needed
- Diagrams to create
- Videos to produce

## Output Format

```markdown
# [Product Name] - Product Page Content

## SEO Elements
- Title: [generated title]

- Primary Keyword: [keyword]
- Secondary Keywords: [list]

## Page Content
[Full page content organized by sections]

[Key ways this page stands out from competitors]
```
