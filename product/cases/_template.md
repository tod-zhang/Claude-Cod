## 示例：已填写的案例

### 信息收集

| 问题 | 回答 |
|------|------|
| 产品名称 | Flywheel |
| 客户行业 | Power Generation |
| 客户国家 | USA |
| 可否透露客户名称？ | 否 |
| 匿名描述 | Leading US generator set manufacturer |
| 材料 | Ductile iron QT450-10 |
| 重量 | 850 lbs |
| 尺寸 | 36" diameter, 4" thickness |
| 关键公差 | G6.3 balance, 0.003" runout |
| 年订单量 | 200 pcs |
| 合作时长 | 3 years |
| 客户之前的问题 | Previous supplier had long lead times (12 weeks) |
| 项目挑战 | Tight balance requirements for low-vibration operation |
| 我们的解决方案 | Optimized gating design for uniform mass distribution |
| 最终结果 | Lead time reduced to 4 weeks, 100% balance spec compliance |

### 产品页简短版输出

```markdown
## Project Example

**Application:** Diesel generator flywheel
**Customer:** Leading US generator set manufacturer
**Specs:** Ductile iron, 36" dia, 850 lbs, G6.3 balance
**Result:** 200 pcs/year, lead time reduced from 12 to 4 weeks

```
