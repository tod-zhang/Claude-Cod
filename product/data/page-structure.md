## Section Guidelines

### Hero Section
- **H1**: Include primary keyword + differentiator
- **Image**: High-quality, custom product photo (not stock)
- **Value Prop**: 2-3 sentences max, focus on benefits
- **Features**: 3-5 bullet points
- **CTA**: Primary action (Request Quote) + Secondary (Call)

### Trust Bar
Position immediately after hero to build credibility:
- Certification logos (ISO, AS9100, etc.)
- Years in business
- Made in USA (if applicable)
- Review rating/count
- Key clients (logos if permitted)

### Product Overview
- Clear, jargon-free definition
- Benefits-focused (not just features)
- 100-200 words
- Include relevant keywords naturally

### Specifications Table
Required elements:
- Materials available
- Size/dimension ranges
- Tolerances achievable
- Surface finish options
- Operating conditions
- Standards compliance

### Manufacturing Process
- Show your unique approach
- 4-6 steps with descriptions
- Include quality control points
- Use images or process video
- Differentiate from competitors

### Industries & Applications
- 4-6 key industries
- Specific use case examples
- Link to detailed application pages
- Include industry-specific images

### Why Choose Us
- 3-4 key differentiators
- Specific proof points (numbers, facts)
- Customer testimonial
- Certification badges

### FAQ Section
- 5-8 common questions
- Use actual customer questions
- Include long-tail keywords

### CTA Section
- Clear call to action
- Multiple contact options
- Reduce friction (no obligation)
- Set expectations (response time)

## Mobile Considerations

```
Mobile Layout Priority:
1. H1 + Value Prop (visible without scroll)
2. Primary CTA (sticky or prominent)
3. Trust signals
4. Key specs (collapsible)
5. Process overview
6. FAQ (accordion)
7. Secondary CTA
```

### Mobile Best Practices
- Sticky header with CTA
- Collapsible sections for specs
- Tap-to-call phone numbers
- Swipeable image galleries
- Accordion FAQs
- Large touch targets (48px min)

## Internal Linking Strategy

### Outbound Links (from this page)
- Related products (3-4)
- Material guide pages
- Process/capability pages
- Application/industry pages
- Contact/quote page

### Inbound Links (to this page)
- Homepage (if featured)
- Category page
- Blog posts mentioning product
- Case studies
- Application pages

### Anchor Text Variety
- Exact match: "custom metal coupling"
- Partial match: "metal couplings"
- Branded: "our coupling products"
- Generic: "learn more", "view details"
- Long-tail: "stainless steel couplings for food processing"

## Image Requirements

| Image Type | Dimensions | Format | Notes |
|------------|------------|--------|-------|
| Hero | 1200x800px | WebP/JPG | High quality, custom |
| Thumbnail | 400x400px | WebP | Square crop |
| Process | 800x600px | WebP/JPG | Show actual work |
| Application | 600x400px | WebP/JPG | Industry context |
| Icons | 64x64px | SVG | Simple, consistent |

### Alt Text Format
```
Bad:  alt="image1"
Bad:  alt="coupling"
Good: alt="Custom stainless steel coupling manufactured by [Company]"
Good: alt="CNC machining process for precision metal couplings"
```
