# Image Plan: 为什么「给 AI 上下文」比「写提示词」更重要

## Image 1
- **Placement:** After intro (before first H2)
- **Type:** Infographic
- **Description:** A split comparison showing "87% use AI" on one side (large crowd of icons) vs "13% get results" on the other (small highlighted group). Visual emphasis on the gap between usage and effectiveness.
- **Alt Text:** 信息图展示 87% 的创作者使用 AI 但只有 13% 获得有意义结果的差距

## Image 2
- **Placement:** After H3 "提示词是指令，上下文是环境"
- **Type:** Diagram
- **Description:** A visual comparison showing two scenarios: Left side shows a simple prompt arrow pointing to AI with generic output. Right side shows the same prompt surrounded by context bubbles (audience, goals, industry, constraints) pointing to AI with tailored output. Demonstrates how context transforms the same prompt into better results.
- **Alt Text:** 对比图：同样的提示词配合不同上下文产生完全不同的 AI 输出结果

## Image 3
- **Placement:** After H3 "方法一：语音输入，3-4 倍速度"
- **Type:** Infographic
- **Description:** Speed comparison visualization: typing icon at 40 WPM vs speaking icon at 150 WPM, with a 3-4x multiplier highlight. Include a visual showing how voice input naturally produces more detailed context (longer text blob) vs typed input (shorter, condensed text).
- **Alt Text:** 语音输入速度对比：说话每分钟 150 字比打字每分钟 40 字快 3-4 倍

## Image 4
- **Placement:** After H3 "方法二：让 AI 采访你"
- **Type:** Diagram / Illustration
- **Description:** A conversation flow diagram showing: User gives initial request -> AI asks clarifying questions (who is your audience? what constraints? where will this be used?) -> User provides answers -> AI produces tailored output. Visual emphasis on the Q&A loop that builds context.
- **Alt Text:** 采访法流程图：AI 通过提问挖掘用户上下文后产出更精准的结果

## Image 5
- **Placement:** After H3 "好上下文的三个特征"
- **Type:** Infographic
- **Description:** Three-part visual showing the characteristics of good context: 1) "具体" with a target icon (direct relevance), 2) "精简" with a filter icon (only necessary info), 3) "可执行" with an action/play button icon (AI can act on it). Include a before/after example: vague "B2B企业" vs specific "卖工业包装机给食品厂采购经理".
- **Alt Text:** 高质量上下文的三个特征：具体、精简、可执行，配合实际示例对比
