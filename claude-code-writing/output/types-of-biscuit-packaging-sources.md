## General Knowledge (No Citation Needed)

- Flow wrap/pillow pack is the most common biscuit packaging format
- Tray and sleeve packaging uses plastic or cardboard trays with film covers
- Stand-up pouches feature bottom gussets for upright display
- Box and carton packaging uses cardboard construction
- Tin packaging provides premium/luxury positioning
- BOPP (Biaxially Oriented Polypropylene) is a standard packaging material
- Metallized films add aluminum layer for barrier enhancement
- Laminates combine multiple materials (paper/OPP/PE, etc.)
- HFFS machines use horizontal orientation; VFFS uses vertical
- European PPWR regulation requires recyclable packaging by 2030
- Mondelez and Duni Group sustainability partnerships (June 2024, March 2024)
