## Part 1: About Us (Brief Introduction)

**Industry**: Cable Management / Electrical Infrastructure Manufacturing

**Core Business**: Manufacturing of cable trays and complete cable management solutions. **Target Markets**:

## Part 2: Target User Analysis

### User Type 1: The Complete Beginner

| Attribute | Description |
|-----------|-------------|
| **Industry** | New electrical apprentices, junior project coordinators, building maintenance trainees, business students studying construction |
| **Knowledge Level** | Novice (0-6 months exposure) |
| **Goals** | Understand what cable trays are and why they're needed; learn basic cable management terminology; understand different tray types |

**What They Already Know**:
- Buildings have electrical wiring
- Cables need to be organized and protected
- Different materials exist for construction (steel, aluminum, plastic)
- Basic awareness of electrical safety importance

**What They Need to Learn**:
- What cable trays are and their purpose
- Different types of cable trays and their applications
- Basic installation concepts
- Industry terminology (ladder tray, solid bottom, etc.)
- Safety and code compliance basics

**Questions They Search For**:
- "What is a cable tray?"
- "Cable tray vs conduit difference"
- "Types of cable trays explained"
- "Cable tray basics for beginners"
- "Why use cable trays?"
- "Cable tray installation guide"

**Typical Roles/Titles**:
- Electrical apprentice
- Junior project coordinator
- Facilities maintenance trainee
- Construction intern

**Pain Points & Challenges**:
- Overwhelmed by product variety and specifications
- Difficulty understanding technical datasheets
- Uncertain how to specify or order the right products
- Lack confidence discussing requirements with engineers

**Content Preferences**:

| Aspect | Preference |
|--------|-----------|
| **Reading Time** | 5-8 minutes |
| **Device** | Mobile-friendly (often researching on-the-go) |
| **Content Length** | 800-1,200 words |
| **Format** | Visual aids, diagrams, comparison charts |

**Decision Factors**:
- Clear explanations without overwhelming detail
- Real-world examples and applications
- Step-by-step learning progression
- Visual representations of products

**Writing Approach**:

| Element | Recommendation |
|---------|----------------|
| **Tone** | Friendly, encouraging, patient |
| **Style** | Conversational; use analogies and real-world examples |
| **Complexity** | Simple sentences; define all jargon; avoid acronyms or explain them immediately |
| **Structure** | Start with "why it matters"; build concepts step-by-step |
| **Word Choice** | Everyday language; assume no prior knowledge |

**Audience-Specific Guidelines**:

- Use everyday analogies: "Think of cable trays like highway systems for cablesâ€”they provide organized routes for wiring..."
- Connect to familiar environments: "The cable trays above drop ceilings in offices, the wire mesh systems in data centers..."
- Provide simple visual descriptions
- Celebrate understanding and progress

- Don't assume knowledge of electrical codes or engineering
- Don't use unexplained acronyms (NEC, NEMA, UL)
- Don't overwhelm with specification details

| **Knowledge Level** | Basic to Intermediate (6 months - 2 years exposure) |

**What They Need to Learn**:

- Comparison criteria between tray types and materials
- Understanding load ratings and fill calculations
- Code compliance requirements
- Cost-benefit analysis of different options

**Questions They Search For**:
- "Ladder tray vs solid bottom comparison"
- "Cable tray sizing guide"

**Typical Roles/Titles**:

- General contractor

**Pain Points & Challenges**:
- Balancing cost, performance, and code compliance
- Evaluating supplier claims and technical specifications
- Managing material substitutions without compromising quality
- Ensuring proper load capacity for cable requirements

**Content Preferences**:

| Aspect | Preference |
|--------|-----------|
| **Reading Time** | 6-10 minutes |
| **Device** | Desktop (during work hours) |
| **Content Length** | 1,200-1,800 words |
| **Format** | Comparison tables, selection guides, checklists |

**Decision Factors**:
- Practical selection criteria and decision frameworks
- Comparison data (cost, load capacity, corrosion resistance)
- Code compliance guidance
- Supplier evaluation criteria

**Writing Approach**:

| Element | Recommendation |
|---------|----------------|
| **Tone** | Professional, practical, direct |
| **Style** | Action-oriented; focus on decision-making frameworks |
| **Complexity** | Moderate; can use industry terms with brief clarification |
| **Structure** | Problem-solution format; comparison tables; numbered lists |
| **Word Choice** | Industry-standard terms; occasional technical vocabulary |

**Audience-Specific Guidelines**:

- Provide selection frameworks: "When choosing a cable tray type, consider these factors..."
- Use comparison tables for materials and tray types
- Offer concrete criteria: "For corrosive environments, consider..."
- Include code compliance guidance (NEC, NEMA references)

- Don't over-explain basic concepts they already know
- Don't provide deep technical details without practical relevance
- Don't overwhelm with options without clear recommendations

| **Knowledge Level** | Intermediate to Advanced (2-5 years experience) |

**What They Already Know**:

**What They Need to Learn**:

- Troubleshooting methods for specific issues
- New products and technologies
- Code updates and compliance details
- Support system design optimization

**Questions They Search For**:
- "Cable tray support spacing requirements"
- "NEC cable tray fill calculations"
- "Cable tray grounding methods"
- "Expansion joint installation cable tray"
- "Cable tray load capacity chart"
- "Fiberglass cable tray installation guide"

**Typical Roles/Titles**:
- Electrical contractor
- Cable tray installer
- MEP engineer
- Electrical superintendent

**Pain Points & Challenges**:
- Ensuring code compliance on complex installations
- Managing field modifications and adaptations
- Coordinating with other trades in tight spaces
- Troubleshooting load and support issues

**Content Preferences**:

| Aspect | Preference |
|--------|-----------|
| **Reading Time** | 8-12 minutes |
| **Device** | Tablet or mobile (on job site) |
| **Content Length** | 1,500-2,500 words |
| **Format** | Technical procedures, load charts, installation diagrams |

**Decision Factors**:
- Detailed technical guidance and procedures
- Code references and compliance requirements
- Specification ranges and load capacities
- Installation best practices

**Writing Approach**:

| Element | Recommendation |
|---------|----------------|
| **Tone** | Technical, precise, matter-of-fact |
| **Style** | Procedural; detailed technical guidance |
| **Complexity** | High; assume technical competence; include specifications |
| **Structure** | Technical procedures; specifications; code references |
| **Word Choice** | Full technical vocabulary; use industry shorthand freely |

**Audience-Specific Guidelines**:

- Provide precise specifications: "Support spacing shall not exceed 5 ft for ladder tray per NEC 392.30"
- Include troubleshooting logic: "If deflection exceeds 1/200 of span, reduce support spacing or..."
- Use technical vocabulary freely (fill ratio, derating, bonding, etc.)
- Reference applicable codes (NEC, NEMA VE 1, UL 2277)

- Don't over-simplify technical concepts
- Don't avoid necessary technical terminology
- Don't omit critical safety or code requirements
