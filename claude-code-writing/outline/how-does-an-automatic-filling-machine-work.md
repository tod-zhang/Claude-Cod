5-stage process:
1. Infeed - containers enter via conveyor
2. Positioning - photoelectric sensors detect, positioning device holds
3. Filling - nozzles dispense using selected method
4. Nozzle retraction - anti-drip mechanism
5. Discharge - to capping/labeling
